Diagramas de flujo

Un diagrama de flujo es una representación gráfica de un proceso o sistema que utiliza símbolos estandarizados para mostrar los pasos, decisiones y flujos de trabajo. Es una herramienta visual que facilita la comprensión, comunicación y análisis de procesos complejos, ya que muestra de forma clara cómo se desarrollan y conectan las diferentes etapas.

Elementos principales de un DFD:

  • Entidad externa: Representa los actores externos al sistema que interactúan con él, como usuarios, organizaciones o sistemas externos. Se suelen dibujar como rectángulos.
  • Proceso: Representa una actividad o tarea que transforma datos de entrada en datos de salida. Se dibuja como un círculo u óvalo y se etiqueta con un nombre que describe la acción.
  • Almacén de datos: Representa un lugar donde se almacenan datos dentro del sistema, como una base de datos, archivos o documentos. Se representa como un rectángulo abierto o líneas paralelas.
  • Flujo de datos: Indica cómo los datos se mueven entre entidades, procesos y almacenes de datos. Se representa con flechas etiquetadas que describen el tipo de datos que se transfieren.
