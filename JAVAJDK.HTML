Instalacion de Java(JDK)

Visitar la página oficial de Oracle Abre tu navegador y ve a la página oficial de descargas de Oracle:

instala aqui

 


Busca la última versión disponible del Java Development Kit (JDK).


Haz clic en la opción de descarga para la versión que quieras instalar (ej. Java 21 o Java 17, dependiendo de tus necesidades).

 


Elegir el sistema operativo En la página de descargas, selecciona el archivo correspondiente a tu sistema operativo

  • Windows: .exe o .msi
  • macOS: .dmg
  • Linux: .tar.gz o .rpm.

Lee y acepta el acuerdo de licencia de Oracle (si es necesario). Haz clic en el enlace de descarga para iniciar la descarga del archivo.


Guarda el archivo descargado en tu computadora, en un lugar fácil de ubicar (como la carpeta de Descargas).


Una vez descargado, procede con la instalación siguiendo las instrucciones específicas para tu sistema operativo.
