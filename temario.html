Temario

1. Algoritmo

Un algoritmo es un conjunto de instrucciones bien definidas que describen una secuencia de pasos a seguir para resolver un problema específico. Es una herramienta fundamental en la programación y en la resolución de problemas en general.

2. Diagrama de flujo (DFD)

Un diagrama de flujo de datos (DFD) es una representación gráfica que muestra cómo los datos fluyen a través de un sistema. Utiliza símbolos estandarizados para representar procesos, entradas, salidas y almacenamientos de datos. Los DFD son útiles para analizar y entender cómo funciona un sistema.

3. Pseudocódigo

Un pseudocódigo es una representación simplificada y semiestructurada de un algoritmo que utiliza un lenguaje similar al lenguaje humano, combinado con estructuras básicas de programación. No sigue una sintaxis estricta de ningún lenguaje de programación en particular, lo que lo hace fácil de leer y comprender tanto para programadores como para personas sin experiencia en programación.

4. Código

El código es la implementación real de un algoritmo en un lenguaje de programación específico. Es un conjunto de instrucciones que una computadora puede ejecutar para realizar tareas concretas. Los programadores escriben código siguiendo la sintaxis de un lenguaje de programación como Python, Java, C++, entre otros.
