Tabla Interactiva de Algoritmos en Java

Algoritmo Descripción Ejemplo Complejidad
Bucle for Se utiliza para iterar un bloque de código un número determinado de veces. for (int i = 0; i < 10; i++) { System.out.println(i); } Baja
Bucle while Permite repetir un bloque de código mientras una condición sea verdadera. while (contador < 10) { contador++; } Baja
Condicional if-else Evalúa una condición lógica y ejecuta diferentes bloques de código según el resultado. if (x > 0) { System.out.println("Positivo"); } else { System.out.println("Negativo"); } Baja
Switch Permite seleccionar entre múltiples opciones basadas en un valor. switch (opcion) { case 1: System.out.println("Uno"); break; } Moderada
Recursión Llama a una función dentro de sí misma para resolver problemas divididos en subproblemas. int factorial(int n) { return (n <= 1) ? 1 : n * factorial(n - 1); } Alta
