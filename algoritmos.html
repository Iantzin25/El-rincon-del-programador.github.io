Algoritmo cualitativo

Un algoritmo cualitativo es un conjunto de pasos o procedimientos diseñados para analizar, procesar o resolver un problema utilizando datos no numéricos o cualitativos. A diferencia de los algoritmos cuantitativos, que se basan en cálculos numéricos y fórmulas matemáticas, los cualitativos trabajan con información descriptiva, categórica o subjetiva.

Agarrar un vaso Entrada
Poner el vaso bajo la llave Proceso
Abrir la llave Proceso
Llenar el vaso Proceso
Quitar el vaso Salida

Algoritmo secuencial

Un algoritmo secuencial es un tipo de algoritmo en el que las instrucciones se ejecutan una tras otra, de forma ordenada y sin desviaciones o saltos en el flujo del programa. Es el tipo más básico de algoritmo y representa procesos donde no se requieren decisiones (condicionales) ni repeticiones (bucles).

Características principales de un algoritmo secuencial

  • Ejecución lineal: Las instrucciones se ejecutan en el orden en que están escritas, de arriba hacia abajo.
  • Sin bifurcaciones: No hay decisiones o ramificaciones que alteren el flujo de las instrucciones.
  • Sin repeticiones: No incluye bucles ni ciclos; cada instrucción se ejecuta una sola vez.

Algoritmo Secuencial: Cálculo del Área

Pasos Descripción
1  Leer la base del rectángulo
2  Leer la altura del rectángulo
3  Calcular el área como base × altura
4  Mostrar el resultado del área

Representación de algoritmos:

  • Pseudocódigo: Descripción textual estructurada.
  • Diagramas de flujo: Representación gráfica con símbolos.
  • Lenguajes de programación: Implementación concreta en código.
