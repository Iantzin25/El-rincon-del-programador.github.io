La palabra "System" es un método, un grupo de instrucciones dentro de una clase.

Declaración de variables y tipos de datos

En Java, existen ocho tipos de datos primitivos:

  • Enteros: byte, short, int, long
  • Punto Flotante: float, double
  • Carácter: char
  • Booleano: boolean

Entrada de datos con Scanner

En Java, la clase Scanner se utiliza para leer datos ingresados por el usuario desde el teclado. Es una herramienta muy útil para interactuar con los usuarios en programas de consola.

Se debe instanciar un objeto de la clase Scanner, que se vincula con System.in para recibir datos del teclado.

Métodos

Un método es un bloque de código que realiza una tarea específica y que puede ser reutilizado. Los métodos son fundamentales en Java, ya que permiten estructurar el código en funciones separadas, mejorando la organización y la reutilización.

  • Modificadores de acceso: Determinan la visibilidad del método (public, private, protected).
  • Tipo de retorno: Indica qué tipo de dato devuelve el método (int, void, String, etc.). Si no devuelve nada, se usa void.
  • Nombre del método: Una etiqueta que identifica al método (debe ser descriptiva).
  • Parámetros: Datos que el método necesita para realizar su tarea. Se declaran dentro de paréntesis.
  • Cuerpo del método: El bloque de código que define lo que hace el método, encerrado entre llaves {}.
